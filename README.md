# GuideLight

GuideLight is a Figma UI/UX prototype designed to improve accessibility for visually impaired users by simplifying access to important services and documents.

## About the Project

GuideLight helps visually impaired users perform daily tasks more independently through a simple and accessible interface.

## Features

### 📄 Document Analysis
Users can:
- Upload a government document or image.
- Capture a photo using the camera.
- Receive a simplified analysis of the document.

### 🗣️ Voice Accessibility
The interface is designed with accessibility in mind and supports voice-guided interactions for visually impaired users.

### 📍 Guidance & Support
The app helps users:
- Find organizations and support centers.
- Know what documents are required before visiting.
- Contact specialists for assistance.
- Navigate using voice-guided directions.
